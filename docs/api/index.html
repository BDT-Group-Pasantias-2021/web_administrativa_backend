Clases

commentClass La clase commentClass se utiliza como un punto de interacción con los comentarios y reacciones que se encuentran guardados en la base de datos. Contiene métodos de creación, modificación y eliminado, así como búsqueda básica por autor y noticia.
noticeClass noticeClass es utilizada para crear un nuevo objeto noticia el cual almacena las funciones y store procedures correspondiente tanto para búsquedas y su edición, creación y eliminación
PDOClass Clase de acceso a datos a través de PDO para cualquier base de datos.
searchAction Ejecuta una función de php de acuerdo a los parámetros recibidos por el método $_GET en su URL para realizar diversas acciones.
userClass Es la clase que se encarga de realizar los llamados a todas aquellas Funciones y Stored Procedures relacionadas a la tabla de Usuarios
